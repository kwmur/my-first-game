Star Attack


[起動方法]
　・ファイル「StarAttack.exe」を実行して下さい。
　・「d3dx9_43.dllが見つからなかったため、このアプリケーションを開始できませんでした。」
　　というメッセージが表示された場合は、「DirectX Runtime\Redist」フォルダ内の
　　「DXSETUP.exe」を実行し、DirectX ランタイム コンポーネントをインストールして下さい。


[操作方法]
　・[Esc]キー　　　　　ゲーム終了
　・[w][a][s][d]キー　プレイヤーの移動
　・マウス　　　　　　　カメラ移動
　・左クリック　　　　　弾を発射


[動作環境]
　・OS - Microsoft Windows XP
　・ビデオカード - DirectX 9.0c 対応
　・空きメインメモリ - 100MB以上
　・画面解像度 - 1024*768以上
